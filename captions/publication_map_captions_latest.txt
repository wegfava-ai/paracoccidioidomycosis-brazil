Map 1. Current predicted suitability
------------------------------------
Current model-predicted environmental suitability for probable acute/subacute paracoccidioidomycosis in Brazil. Values represent model-predicted annualized suitability rates per 100,000 inhabitants using the M3 climate-soil-elevation negative binomial model trained on 1998-2024 municipality-year data. The map should be interpreted as relative suitability, not observed incidence.
Recommended use: main_or_supplementary

Supplementary Map S1. Current suitability classes
-------------------------------------------------
Quantile-based classes of current model-predicted suitability for probable acute/subacute paracoccidioidomycosis. Classes summarize the distribution of current predicted suitability across Brazilian municipalities.
