# ENSO and probable acute/subacute paracoccidioidomycosis in Brazil

This repository folder contains the reproducible outputs for the exploratory ENSO analysis and municipal incidence maps of probable acute/subacute paracoccidioidomycosis (PCM) in Brazil.

## Scope

- Outcome: probable acute/subacute PCM, defined using DATASUS records with age <= 20 years.
- Period: 1998-2024.
- Spatial unit: Brazilian municipalities.
- Temporal exposure: annual mean Oceanic Niño Index (ONI).
- ENSO classification: El Niño if annual mean ONI >= 0.5; La Niña if annual mean ONI <= -0.5; Neutral otherwise.

## Folder structure

```
scripts/                 R scripts used to generate the analyses and figures
data/                    Lightweight processed RDS data used by the figures
tables/                  CSV tables generated by the analyses
figures/enso/            ENSO and PCM burden figures
figures/incidence_maps/  Municipal incidence maps
logs/                    Execution reports from the validated scripts
docs/                    Data dictionary and methods notes
```

## Main scripts

1. `24_analyze_enso_impact_on_pcm_incidence.R`
   - Builds the national annual PCM + ENSO dataset.
   - Corrects the national population denominator using the widest valid IBGE municipal population file.
   - Fits exploratory ENSO regression models.
   - Creates ONI and ENSO/PCM figures.

2. `24b_create_pcm_incidence_period_maps.R`
   - Creates annualized municipal incidence maps for the overall period and subperiods.
   - Uses the same validated population denominator logic.
   - Uses a safe local state boundary layer to avoid heavy geometry operations.

3. `24d_create_combined_oni_pcm_burden_figure.R`
   - Creates a single combined figure showing annual ONI, annual PCM case counts, and annual incidence.

## Methodological note

The ENSO analysis is descriptive and exploratory. ENSO is a year-level exposure, so the primary analysis uses annual national counts and population offsets. Results should not be interpreted as causal evidence.

The incidence maps show annualized municipal incidence per 100,000 person-years. Municipalities without records should not be interpreted as areas without risk; they may reflect low incidence, underdiagnosis, underreporting, or limited surveillance.

## Data sharing note

This package is designed to avoid sharing raw individual-level DATASUS records. It includes processed aggregate tables, figures, logs, and scripts needed for transparency and reproducibility.
